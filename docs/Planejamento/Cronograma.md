# Cronograma

O cronograma funciona como um instrumento para planejar tarefas, recursos e prazos de um projeto de maneira visual, neste caso, por meio de tabelas. Os cronogramas a seguir foram elaborados considerando as datas e prazos estabelcidos no enunciado disponibilizado no classroom da EngNet.
